☕ Camper Cafe Menu

A simple web project — a cafe menu built using HTML and CSS as part of the FreeCodeCamp course.

📌 About the Project

This is a static web page that displays a cafe menu with coffee and desserts.  
The project helps to understand the basics of HTML structure and CSS styling.

🚀 Technologies

- HTML5
- CSS3

📂 Features

- Coffee menu with prices
- Dessert section
- Clean and simple layout
- Background image (coffee beans)
- Centered design
- Adaptation to screen sizes
